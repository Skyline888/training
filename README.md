First, run the development server:


День 1: Верх (Акцент на ширину — Плечі та Спина)
Це ключовий день для створення V-форми.
Тяга верхнього блоку до грудей (широкий хват): 3 підходи по 10-12 пов. (Будуємо ширину спини).
Жим гантелей стоячи/сидячи: 3 підходи по 8-10 пов. (База для маси плечей).
Махи гантелями в сторони: 4 підходи по 12-15 пов. + пауза у верхній точці. (Саме ця вправа робить плечі ширшими).
Шраги з гантелями (з паузою): 3 підходи по 12 пов. (Верх трапеції для потужного вигляду).
Віджимання на брусах (або жим лежачи): 3 підходи по 8-10 пов. (Груди та трицепс).
День 2: Низ (Потужність та база)
Гакк-присідання (або фронтальні): 3 підходи по 8-10 пов. (Акцент на квадрицепс).
Румунська тяга: 3 підходи по 10-12 пов. (Задня частина стегна та сідниці).
Розгинання ніг у тренажері: 3 підходи по 12-15 пов.
Підйоми на литки стоячи: 4 підходи по 15-20 пов. з фіксацією вгорі.
Планка або підйом ніг у висі: 3 підходи до паління. (Для корекції вісцерального жиру та міцного кору).

День 4: Верх (Акцент на деталізацію та об'єм)
Тяга штанги/гантелі в нахилі: 3 підходи по 10 пов. (Товщина спини).
Жим гантелей на похилій лаві (30-45°): 3 підходи по 10-12 пов. (Верх грудей візуально підіймає торс).
Тяга до підборіддя (широким хватом): 3 підходи по 12 пов. (Ще один удар по середній дельті для ширини плечей).
Face Pulls (Тяга каната до обличчя): 3 підходи по 15 пов. (Здоров'я плечей та задня дельта).
Суперсет: Біцепс + Трицепс: по 3 підходи для об'єму рук.
День 5: Низ (Доопрацювання)
Жим ногами: 3 підходи по 12-15 пов.
Згинання ніг сидячи/лежачи: 3 підходи по 12-15 пов.
Випади з гантелями: 3 підходи по 12 кроків на кожну ногу.
Підйоми на литки сидячи: 3 підходи по 15-20 пов
